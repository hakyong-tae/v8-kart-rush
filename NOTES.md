# V8 Kart Rush — 구조 분석 노트

카트라이더 스타일 3D 아케이드 카트 레이서. **모든 코드 직접 구현**, 에셋은 CC0(Kenney Racing Kit)만 사용 — 저작권 이슈 없음, Verse8 업로드 가능.

## 핵심 기능

- **부스터 게이지 (카트라이더 코어 루프)**: 드리프트로 게이지 충전 → 풀차지 시 E/Ctrl로 부스터 발동 (2.3초, 최고속 41). 드리프트 릴리즈 시 짧은 순간부스트(2단계)도 있음
- **스타트 부스터**: 카운트다운 중 ↑를 누르면 기 충전(3초 풀차지) — 그린존(35~92%)에서 GO하면 비례 부스터, **과충전(92%+)이면 엔진 폭발(스핀 패널티)**. HUD에 존 표시 게이지
- **4가지 모드**:
  - **싱글 스피드전**: 타임어택 + **1위 고스트가 같이 달림** (온라인=글로벌 1위 리플레이, 오프라인=내 베스트). 기록은 코스별 글로벌 리더보드 등록
  - **싱글 아이템전**: AI 3명과 아이템 레이스 (러버밴딩으로 접전 유지, 순위 결과 화면) — 리더보드 미반영
  - **멀티 스피드전 / 멀티 아이템전**: 룸 기반 최대 8인, 방장이 모드 선택
- **가드레일 벽**: 트랙 양쪽 벽 충돌 — 충격각 비례 감속+바운스, 강충돌 시 드리프트 캔슬
- **캐릭터 × 카트 분리 로스터**: 캐릭터 5종(외형+보너스) × **카트 4종 — 전부 다른 차체** (스파크 R=레드 F1 / 터보 G=그린 핫해치 / 맥스 O=오렌지 머슬 / 코멧 X=블루 퓨처카, Kenney Car Kit). **최종 스탯 = 100 + 캐릭터 포인트 + 카트 포인트 (덧셈)**, 타이틀 화면에 회전 3D 조합 프리뷰 + 최종 스탯 바. 카트별 라이더 탑승 위치(`riderPos`) 정의
- **아이템 2슬롯** (카트라이더식): 부스트 🚀 / 미사일 🎯 / **바나나 🍌**(밟으면 스핀) / **폭탄 💣**(전방 투척, 2.2초 후 반경 4.5 폭발 — 자폭 주의) / **실드 🛡️**(1회 방어) / **번개 ⚡**(전원 스핀)
- **치비 라이더** 캐릭터 탑승, 뭉게구름, 부스터 시 스피드라인+FOV 펀치, 칩튠 BGM (WebAudio 시퀀서)
- 부스트 패드, 아이템 박스, 오프로드 감속(코스별 상이), 역주행 경고, R 리셋
- **오프라인 폴백**: 서버 미연결 시 타임어택 + localStorage 리더보드로 자동 전환

## 아키텍처

### 트랙 시스템 (track.ts)
- 코스 = 2D 제어점 배열 → `THREE.CatmullRomCurve3` (closed) → 1000개 샘플 (pos/tangent/normal)
- 도로/연석/센터라인 = 스플라인 양쪽 lateral offset으로 스트립 메시 생성 (vertex color)
- **트랙 좌표계**: 카트마다 `trackIdx`(최근접 샘플, 직전 인덱스 주변 ±30 로컬 탐색 = O(1)) + lateral offset
- **체크포인트**: 트랙을 8구간 분할, 순차 통과만 인정(`cpTotal` 단조 증가) → 코스컷 방지, 후진 시 롤백
- 랩 = cpTotal이 8의 배수+1 도달 시 (스타트라인 재통과)

### 카트 물리 (kart.ts)
- 종방향: 포화 가속(`ACCEL*(1-v/max)`), 잔디 시 max 10/드래그 강화, 패드부스트 max 37, 부스터 max 41
- 조향: 속도 의존 회전율, **velDir(이동방향)이 heading(차체방향)을 grip 속도로 추적** → 드리프트 시 grip 8→1.55로 강한 슬라이드. 카운터스티어로 드리프트 각 조절
- **부스터 게이지**: 드리프트 중 0.3/s 충전(스티어 강도 보너스) — 드리프트 끊겨도 게이지는 유지(누적), 풀차지 → `fireBooster()` 2.3초
- 순간부스트: 드리프트 0.7s/1.5s 차징 후 해제 시 0.45s/0.85s 짧은 부스트
- **빙판(ice surface)**: grip 8→3.2 (드리프트 시 1.1) — 아이스 밸리 전용
- **가드레일**: `wallDist = halfWidth + shoulder` 초과 시 클램프 + 법선 성분 반사(바운스 0.35) + 충격각 비례 감속, 강충돌 시 드리프트 캔슬

### 멀티플레이 (net.ts + server.js)
- Verse8 `@agent8/gameserver`: `GameServer.getInstance()` 직접 사용 (React 훅 없이)
- 룸: 공개방 `q-{courseId}`, 코드방 `r-{code}-{courseId}`; 방장 = `$users[0]`
- 룸 상태 키를 **flat하게** (`p_{account}`, `fin_{account}`) → updateRoomState 얕은 병합에서 동시쓰기 충돌 방지
- 위치 동기화: `remoteFunction('updatePos', ..., {throttle: 100})` → 서버 `$room.broadcastToRoom('pos')` 릴레이 (상태 저장 없음) → 수신측 110ms 버퍼 보간 + 1.35x 외삽
- 레이스 시작: `startAt`(서버 epoch) 기준 동기 카운트다운, 클라이언트는 `now()` RTT/2로 시계 오프셋 보정
- 아이템: 피해자 권한(victim authority) — 자기 카트 피격만 자기가 판정 후 브로드캐스트
- **멀티 기록은 리더보드 미반영** (아이템전이라 불공정) — 타임어택만 등록

### 리더보드 (server.js)
- 코스별 글로벌 컬렉션 `times_{courseId}`, 1인 1엔트리 (갱신 시 기존 삭제)
- `orderBy: [{field: 'totalMs', direction: 'asc'}]` — 3랩 합계 오름차순
- 서버 검증: 코스 화이트리스트, 30초~30분 범위, bestLap ≤ total, 닉네임 1~15자

## 실행 방법

```bash
export PATH="$HOME/.nvm/versions/node/v23.11.0/bin:$PATH"
npm install
npm run dev          # http://localhost:3014 (오프라인 모드)
```

### Verse8 서버 배포 (멀티플레이/글로벌 리더보드 활성화)

```bash

# → .env에 VITE_AGENT8_VERSE 생성됨 → npm run dev 재시작하면 온라인 모드
```

배포 전까지는 자동으로 오프라인 모드 (타임어택 + 로컬 기록)로 동작한다.

## 조작

| 키 | 동작 |
|---|---|
| ↑↓←→ / WASD | 가속·브레이크·조향 |
| Shift / Space | 드리프트 (게이지 충전 + 순간부스트) |
| E / Ctrl | 부스터 발동 (스피드전) / 아이템 사용 (아이템전) |
| R | 트랙 복귀 |

## 코스

| 코스 | 난이도 | 특징 |
|---|---|---|
| 써니 서킷 (sunny) | ★ | 완만한 순환, 입문용 |
| 선셋 비치 (beach) | ★ | **오픈 맵** — 가드레일 없는 섬, 모래 위 ~15 속도로 지름길 컷 가능 (부스터로 풀스피드 컷), 부표가 경계 |
| 캐니언 트위스트 (canyon) | ★★ | S자 연속 + 사막 테마 |
| 아이스 밸리 (ice) | ★★ | **빙판 저그립** — 드리프트 제어 연습 |
| 네온 나이트 (neon) | ★★★ | 좁은 도로(11), 테크니컬, 야간+가로등 |

오픈 맵(`open: true`)은 가드레일 메시 생략, 섬(원형 모래) + 바다 평면, 부표/야자수/파라솔 절차 생성 데코를 사용한다. `offroadMax`/`offroadDrag`로 코스별 오프로드 페널티 조절 (비치: 평형 ~15 = 컷 전략 유효, 일반 코스: ~6 = 페널티).

코스 추가 = `courses.ts`에 제어점/패드/테마만 추가하면 끝 (도로/체크포인트/미니맵/데코 자동 생성). 단, `server.js`의 `COURSE_IDS` 화이트리스트에도 추가 필요.

## 고스트 시스템

- 스피드전 주행을 100ms 간격으로 기록 (x, z, heading — 3랩 ≈ 800샘플)
- 신기록 시 서버 `ghosts_{courseId}` 컬렉션에 저장 (1인 1고스트) + localStorage 백업
- `getTopGhost(courseId)`: 글로벌 1위의 고스트 + 카트/캐릭터 조합을 반투명 모델로 재생
- 주의: GLB 텍스처(`Textures/colormap.png`)는 외부 참조라 public/models/Textures/에 함께 배포

## 에셋 라이선스

- `public/models/` — Kenney Racing Kit 2.0 + Car Kit 3.1, **CC0** (퍼블릭 도메인). 라이선스 파일 동봉
- 사운드 — WebAudio 신디사이저로 런타임 합성 (에셋 파일 없음)
- 닌텐도/마리오카트 에셋 일절 미사용 (상표·저작권 리스크 차단)

## 알려진 한계 / 다음 단계
